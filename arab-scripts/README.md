# 🛒 ArabScripts | منصة عرب سكربتس

النسخة التجريبية الأولى (MVP Beta) لمنصة بيع سكربتات وحلول رول بلاي للعبة FiveM. المنصة مبنية بأحدث تقنيات الويب والأمان وتتضمن نظام تراخيص معقد يتصل بسيرفرات اللعبة مباشرة.

## 🚀 التقنيات المستخدمة (Tech Stack)
- **الإطار العام:** Next.js 16 (App Router)
- **قاعدة البيانات:** Prisma ORM + SQLite (جاهزة للتحويل إلى PostgreSQL في الإنتاج)
- **التصميم:** Tailwind CSS v4, Shadcn/ui (Glassmorphism, Dark Theme, RTL)
- **الأمان:** NextAuth (Credentials + JWT)، Bcrypt، Route Middleware

---

## ⚙️ طريقة التشغيل والتثبيت محلياً (Local Setup)

1. **تثبيت الحزم (Install Packages):** 
   ```bash
   npm install
   ```

2. **تجهيز متغيرات البيئة (.env):** 
   انسخ ملف `.env.example` إلى `.env`
   ```bash
   cp .env.example .env
   ```

3. **تجهيز قاعدة البيانات (Database Setup):** 
   سيقوم الأمر التالي بتجهيز الجداول وبناء الملفات الأولية في `dev.db` وحقن بيانات التجربة (Seed):
   ```bash
   npx prisma db push
   npm run seed
   # In this project, seed might be run via: npx tsx prisma/seed.ts
   ```

4. **تشغيل المشروع:** 
   ```bash
   npm run dev
   ```
   > سيصبح السيرفر متاحاً على `http://localhost:3000`

## 📱 تجربة النظام كعميل (User Flow)
1. سجل الدخول بحساب العميل.
2. تصفح المنتجات في `/products`.
3. اضغط (متابعة الشراء) لأي منتج.
4. في شاشة الدفع الآمنة، اضغط "تأكيد الدفع" (نظام محلي للمحاكاة).
5. انسخ **مفتاح الترخيص** الخاص بك من شاشة النجاح وقم بتحميل ملف البرمجة الوهمي.
6. تابع تراخيصك في **لوحة التحكم** `/dashboard`.

## 🛡️ استخدام نظام حماية التراخيص (License API)
لمزامنة السكربت الذي تبيعه مع سيرفر الـ FiveM الخاص بالعميل، أضف الكود الموجود في:
`examples/fivem_license_check.lua`
إلى بداية السكربت المباع. هذا السكربت سيقوم بنداء:
`POST /api/licenses/verify`
للتأكد مما يلي قبل التشغيل:
- الرخصة نشطة.
- الرخصة متطابقة مع نوع المنتج المباع.
- لم يتم تجاوز الحد الأقصى لتفعيلات الخوادم (Server Limit).

> بمجرد انتهاء البيتا، فقط غيّر `DATABASE_URL` في ملف الـ `.env` الخاص بك لعنوان `Postgres` وسيتم نقل المنصة بالكامل للسحابة. بالتوفيق!
